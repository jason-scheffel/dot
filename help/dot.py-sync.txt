usage: dot.py sync [-h] [--no-progress] [--quiet] [--no-git] dotfiles_repo

Sync the dotfiles repo.

positional arguments:
  dotfiles_repo  The path to the dotfiles repo.

options:
  -h, --help     show this help message and exit
  --no-progress  Show progress bar. (default: False)
  --quiet        Show no verbose output. (default: False)
  --no-git       Do not run git commands. (default: False)
