usage: dot.py install [-h] [--no-progress] [--quiet] dotfiles_repo

Install the dotfiles repo.

positional arguments:
  dotfiles_repo  The path to the dotfiles repo.

options:
  -h, --help     show this help message and exit
  --no-progress  Show progress bar. (default: False)
  --quiet        Show no verbose output. (default: False)
